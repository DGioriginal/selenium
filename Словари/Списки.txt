del list[index]-Нужен для того, чтоб удалить значение в списке по индексу
not in/in- Проверка, если ли значение в списке
for index, item in enumerate(list)-Оно возвращяет 2 значения, индек значения и само значение в списке, с помощью чего уменшает код, если нужно например вывести весь список пользователю.
random.choice(list)- Функция возвращяет случаное значение из списка
random.shuffle(list)- Перетасовывает список
list.index('значение')- находит введеное значение по индексу
list.append('значение')- добавляет новое значение в конец списка
list.insert(index, 'значение')- добавляет значение на нужный индекс
list.remove('значение')- удаляет значение из списка, самое первое что увидет, а не все из списка
list.sort()-сортировать список по алфавиту или цыфрам
list.sort(reverse=True)- меняет значения в списке местами
list.sort(key=str.lower)- сортировка значений в списке по алфавиту
list.reverse()- такой же как и list.sort(reverse=True)
tuple([list])- Делает список кортежом
list('word')- все что в скобках, делает в список, каждую букву и пробелы
id()- узнать идентификатор нужного значения
pere = copy.copy(list)- копирует список или словарь полностью для новой переменной, а не просто сыллку.
pere = copy.deepcopy(list)- нужно использовать в том случае если список содержит вложенные списки, в таком случае она копирует всю структуру.
